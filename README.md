# بلوقن نسخ حساب - Copy Profile

**المطور:** @DRR_R2  
**القناة:** @phpandpy  

## 🚀 الأوامر
- رد على شخص وأرسل `.copy` لنسخ حسابه
- `.backup` لحفظ نسخة من حسابك
- `.restore` لاسترجاع حسابك الأصلي

⚠️ لا تنسخ حسابات رسمية - يؤدي إلى حظر.

## 🔧 التثبيت
ضع الملف في مجلد `plugins` وافعله من إعدادات التطبيق.
